## Features

- **Loading Screen** – Displays a spinner while the quiz loads.  
- **Multiple Questions** – Six questions with four options each.  
- **Question Counter** – Shows current question number out of total.  
- **Timer** – 10-second countdown per question.  
- **Progress Bar** – Visual representation of quiz progress.  
- **Score Tracking** – Updates score dynamically for correct answers.  
- **Answer Feedback** – Highlights correct answers in green and wrong answers in red.  
- **Final Result Screen** – Shows total score at the end of the quiz.  
- **Restart Quiz** – Allows users to restart the quiz without refreshing the page.  
- **Responsive Design** – Works on both desktop and mobile using Bootstrap.

## How It Works

1. The quiz starts with a **loading screen**.  
2. Questions are displayed **one by one**, with a 10-second timer for each.  
3. Selecting an answer highlights it as **green (correct)** or **red (wrong)**.  
4. The **score and progress bar** update dynamically.  
5. At the end, the **final score screen** shows your total score, with an option to restart.
